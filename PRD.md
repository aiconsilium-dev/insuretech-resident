# PRD: InsureTech Resident (입주민 앱)

## 개요

아파트 단지 입주민이 세대 내 피해(누수, 화재, 시설사고 등)를 사진과 함께 접수하면,
AI가 TYPE A/B/C로 자동 분류하고 예상 보상금액을 산출하는 모바일 웹 앱.

## 대상 사용자

- 아파트 입주민 (소유자/임차인)
- URL 파라미터로 사용자 식별: `?name=&apt=&dong=&ho=`

## 핵심 기능

### 1. 홈 (대시보드)
- 단지/호수 배지, 인사말
- 긴급 사고접수 CTA 카드
- 내 접수 현황 (TYPE별 건수)
- 빠른 메뉴 4개: 사고접수, 내 접수, 보험안내, 서류관리
- 현장조사 진행 알림 배너

### 2. 사고접수 (3-step)
**Step 1 - 피해유형 선택**
- 6가지: 하자/누수, 화재, 시설사고, 가재도구, 신체부상, 기타

**Step 2 - 상세정보 입력**
- 동/호 (URL 파라미터에서 자동)
- 발생일시, 피해자 구분 (소유자/임차인)
- 사진 3종 (전체, 주변부, 발생부위 상세)
- 누수 시: 기술 소견서 업로드
- 피해 설명 텍스트
- 관리사무소 현장조사 요청 체크

**Step 3 - AI 분석 & 결과**
- 4단계 분석 애니메이션: 사진 분석 → 원인 판단 → TYPE 분류 → 금액 산출
- 결과: TYPE A/B/C 카드, 분석 텍스트, 예상 보상금액
- 접수하기 → 완료 모달

### 3. 내 접수
- 접수 건 리스트 (ID, 유형, 날짜, TYPE 배지)
- 4단계 진행바: 접수 → 검토 → 산정 → 완료
- 탭하면 상세 정보 확장

### 4. 더보기
- 프로필 카드
- 보험안내: CGL, 주택화재, 가재보험 설명
- 서류관리: 건축물대장, 등기부등본, 신분증 상태
- 프로필: 사용자 정보 확인

## TYPE 분류 로직

| 피해유형 | TYPE | 설명 |
|---------|------|------|
| 누수/화재/시설/가재도구 | C | 보험금 산출 대상 |
| 신체부상 | A | 시공사 하자, 하자보수 청구 |
| 기타 | B | 면책 검토 대상 |
